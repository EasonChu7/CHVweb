資訊

* 伺服器版本 : Java 1.16.1

* 伺服器IP : mcchv.space

* 伺服器官網 : 尚未開放

* 伺服器巴哈 : https://forum.gamer.com.tw/C.php?bsn=18673&snA=186452&tnum=1&subbsn=18

* 伺服器Discord : https://discord.gg/9hM5krD

新手引導

  • 使用連線IP:mcchv.space 加入伺服器後 右鍵點選雲天國NPC加入雲天國大廳分流

  • 進入雲天國大廳分流後，可使用鍵盤按鍵/或T開啟聊天室窗並輸入指令/.開啟選單

  • 選單有大大小小諸多功能，若想優先體驗本服RPG打怪樂趣請找到並點擊按鈕分流選擇，選擇RPG分流

  • 進入RPG分流後，請直走前往傳送門，進入傳送門後，後方可以領取基礎裝備與選擇職業(未開放)

  • 接著前往傳送門區並傳送到野外世界探險與蒐集資源，無盡煉獄與不穩定世界之傳送門為隨機傳送；虛無深淵之傳送門為定點傳送

  • 野外可以見到三種怪物(普通士兵、基礎士兵、菁英怪)，打死怪物掉落之素材可以到大廳分流輸入指令/res tp market或右鍵點選主城NPC 市集傳送前往市集兌換物品

  • 接者可以使用指令/.開啟選單，點擊按鈕分流選擇，選擇空島分流

  • 進入空島分流後，可使用指令/.開啟選單後找到空島創建按鈕創建空島或使用指令/sk create <ID>，可以選擇沙漠地形或普通地形

  • 空島相關指令都可以在選單中找到，或查看下方指令教學

不穩定系列

不鏽鋼爐
最低所需材料：煤炭磚1個、鐵磚1個、岩漿100Cml(0.1桶)、不穩定金屬鉻、錳
、鋁、鈦、鎳各1個、不穩定元素硫、矽、磷各1個

最低產出數量：不鏽鋼10個

使用條件：無

機組位置：自行召喚（須購買）

使用方法：
蹲+右：放入素材
蹲+左：退還1次素材
(岩漿不退還)
右：開始反應(材料須多於等於最低所需材料)
左：查看目前材料數量
核分裂反應爐
最低所需材料：不穩定能量元素鈾235 4個(同時也是最大容量)

最低產出數量：不穩定能量核能、化學能共4000CE(40個物品)

使用條件：全程穿著核能保護裝

機組位置：自行召喚(須購買)

使用方法：
蹲+右：放入素材
蹲+左：退還1次素材
(已知問題：退還的素材無法與掉落的重疊，但可以放入反應爐)
右：開始反應(材料須等於最低所需材料)
左：查看目前材料數量
核融合反應爐
最低需求材料：不穩定能量元素氘2、氚3各8個(同時也是最大容量)、不穩定能量熱能、
化學能或核能共2500CE(25個物品)、藍冰25個

最低產出數量：不穩定能量熱能、光能、電能共15000CE(150個物品)

使用條件：全程穿著核能保護裝

機組位置：自行召喚(須購買)

使用方法：
蹲+右：放入素材
蹲+左：退還1次素材
(能量、藍冰不退還)
右：開始反應(材料須多於等於最低所需材料)
左：查看目前材料數量
